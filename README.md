# toSPM

toSPM предназначен для конвертирования пакетов из других дистрибутивов в SPM.

## Поддерживаемые пакетные менеджеры

- [x] Debian/Ubuntu DPKG (используйте `deb`)
- [ ] Arch Linux PACMAN
- [ ] Arch Linux PKGBUILD
- [ ] Fedora/RHEL RPM

## Использование

```
tospm [пакетный менеджер] [имя файла]
```
